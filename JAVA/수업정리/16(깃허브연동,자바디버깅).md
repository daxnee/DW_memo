## 디버깅 하는 이유?
  우리의 업무 효율성 상승
  효율성 -> 칼퇴!


## 디버깅하는 방법>
  추적하고 싶은 코드 번호를 더블 클릭하면 
  콩알 모양이 나옴
  run - debug As - switch - f6 누를때마다 실행
  디버깅 단축키  :  f11

  ★디버깅을 잘 보기 위해서!★
  1. 변수를 막 만든다
  2. 디버깅을 한다.
    why? 변수가 없는 데이터는 디버깅때 무시됨
  3. 변수를 제거한다(불필요한, 중복되는)

  값이 이상할 때 마다 디버깅해서 그 코드부터 다시 고치면 됨(f11)
