### 모든 클래스의 초기값은 NULL이다
String 도 클래스이기 때문에 값을 설정(set)하지 않으면 
출력시 null이 나온다.

```java
Member m = new Member();
		m.setName("토르");
		String name = m.getName();
		System.out.println("m => " + name);
		
//		Member m2 = new Member(); //  위와 다르게 set을 안했으니 => 값은 초기값인 'null'
		Member m2 = m; // 인스턴스화 하지 않고 변수 m을 대입해줌 => 값은 '토르'
		String name2 = m.getName();
		System.out.println("m => " + name);
```
