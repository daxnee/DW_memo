##IP종류 
스벅에서 여러 사람이 쓰는 IP - 사설 IP or Private IP
인터넷 기사가 설치한 인터넷 선에서 나오는 것 - 공인 IP

*인터넷 기사가 설치한 (공인IP) 선에서 개인에게 부여받는 것(사설IP)
밖에 유출되는 IP는 공인임 (사설은 알 수 없는 구조) 

★로컬(local) : 본인이 쓰는 PC를 말함(컴퓨터 설치하면 기본으로 설정되어 있는 주소)
로컬주소 (본인 PC)  : 127.0.01 (한 공간 안에 있는 사람들은 똑같은 주소를 씀) = localhost 라고 적어도 됨
ex) ip => 192.168.0.32 = localhost (=본인 IP를 의미) 

*팀플할때는 각자 다르게 구분하려고 그냥 IP
혼자 개발할땐 로컬IP로 개발함
