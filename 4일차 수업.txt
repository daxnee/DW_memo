***DDL(Data Definition Language) 데이터 정의 언어- 단위 :TTABLE

-DROP TABLE <table_name> : table 삭제
-ALTER TABLE <table_name> : table 수정 (테이블 이름 변결할때 씀)

-DELETE : 데이터 삭제

-INSERT : 데이터 생성

##데이터 조회해보기
Q. 데이터 안에 있는 명단중, 고유값이 7839인 사람의 직업은?
select
job
from emp
where empno = 7839;

Q. 데이터 안에 있는 명단중, 고유값이 7900인 사람의 이름과 직업은?
select
ename, (두개를 조회하려면 콤마로 구분해서 한번에 조회하기)
job
from emp
where empno = 7900; (where은 조건문임) - 원하는 데이터 추출 
*where 지우면 모든 데이터가 추출됨

Q. 모든 데이터를 조회하시오
select
* (모든 데이터를 조회하겠다 ,애스터리스크(?))
from emp;
